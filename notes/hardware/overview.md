# Hardware Overview

## Primary Platform
- Panasonic Toughbook CF-31 (Mk1 rugged laptop)

## Secondary Compute
- Lenovo ThinkCentre Tiny mini desktop

## Embedded / Radio Hardware
- 2 × Heltec WiFi LoRa 32 V3 boards
- RTL-SDR USB receiver
- Rooftop LoRa antenna tuned around 915 MHz
- Additional radio / RF experiment hardware

## Current Goals
The current project direction is to extend the useful life of the Toughbook platform while preserving its ruggedness, portability, and standalone functionality.

Primary goals:
- Keep the Toughbook fully functional as its own computer
- Preserve the ability to boot the Toughbook’s original operating system
- Explore dual-computer operation with a secondary modern x86 computer
- Maintain portability as much as possible
- Prioritize full PC software compatibility

## Constraints
- Keep the Toughbook rugged chassis intact
- Maintain portability
- Allow dual-computer operation if possible
- Prefer practical and serviceable modifications
