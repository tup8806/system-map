# Toughbook Role

## Primary Uses
- portable Debian workstation
- management terminal for the home server
- network diagnostics and troubleshooting laptop
- hardware inspection and power-tuning platform
- portable AI command station

## Important Software
- OpenClaw
- Google Gemini CLI
- Docker
- Git
- SSH / SSHFS
- Tor
- Samba tools
- xscreensaver

## Important Services Enabled at Boot
- SSH server
- Tor
- NetworkManager
- powertop
- smartmontools
- lm-sensors

## Key Capabilities
- remote server access and administration
- network testing and DNS troubleshooting
- container-based experiments with Docker
- disk and hardware health monitoring
- low-power laptop tuning
- privacy-focused networking
- local AI tooling through command-line interfaces

## Related Systems
- Optiplex home server on local network

## Notes
- OpenClaw is installed globally through npm
- Gemini CLI is installed globally through npm
- system package inventory is stored separately in outputs
