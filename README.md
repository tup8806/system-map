# System Map

This repository tracks my systems, important programs, configs, and major changes over time.

## Purpose
- Record installed software and services
- Back up important user configs
- Track system changes
- Make troubleshooting easier
- Keep a nightly update habit

## Main folders
- scripts/  -> collection and backup scripts
- outputs/  -> generated system info
- configs/  -> copied config files
- notes/    -> human notes
